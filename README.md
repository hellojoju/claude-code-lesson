# Claude Code 教程网站

[Claude Code](https://claude.ai/code) 中文教程网站，基于 [claude-howto](https://github.com/luongnv89/claude-howto) 整理汉化。

## 在线访问

```bash
cd website
python3 -m http.server 8081
```

然后打开 http://localhost:8081
