## [ ] 任务 2：基础教程内容编写 - 安装与配置（手把手教程）
- **优先级**：P0
- **依赖**：任务 1
- **描述**：
  - **系统要求检查**（包含验证命令）：
    - macOS：打开终端，运行 `sw_vers` 查看版本（需要 10.15+）
    - Ubuntu/Debian：运行 `lsb_release -a` 查看版本（需要 20.04+/10+）
    - Windows：运行 `wsl --version` 检查 WSL 安装状态
    - Node.js：运行 `node --version` 检查版本（需要 18+）
    - npm：运行 `npm --version` 检查版本
  - **安装方法一：Claude Desktop App**（每个步骤都有截图）：
    - 步骤1：打开浏览器，访问 https://claude.ai/download
    - 步骤2：点击对应操作系统的下载按钮
    - 步骤3：运行下载的安装程序（.dmg/.exe/.deb）
    - 步骤4：安装完成后，打开 Claude Desktop App
    - 步骤5：使用 Anthropic 账号登录（如果没有账号，点击"Sign Up"注册）
    - 步骤6：验证安装成功 - 在终端运行 `claude --version`
  - **安装方法二：命令行安装**（包含每个命令和预期输出）：
    - 步骤1：检查 Node.js 版本
      ```bash
      node --version
      # 预期输出：v18.x.x 或更高
      ```
    - 步骤2：全局安装 Claude Code
      ```bash
      npm install -g @anthropic-ai/claude-code
      # 预期输出：added 1 package in 5s
      ```
    - 步骤3：验证安装
      ```bash
      claude --version
      # 预期输出：claude-code version x.x.x
      ```
  - **国内使用方法一：Claude Code Router**（详细到每个配置步骤）：
    - 步骤1：安装 Claude Code Router
      ```bash
      npm install -g claude-code-router
      ```
    - 步骤2：初始化配置
      ```bash
      ccr init
      # 选择选项 1 进行初始化
      ```
    - 步骤3：打开 Web 界面
      ```bash
      ccr web
      # 浏览器会自动打开 http://localhost:8080
      ```
    - 步骤4：在 Web 界面中点击"添加供应商"
    - 步骤5：选择"从模板导入"，选择你的 API 供应商
    - 步骤6：输入 API 密钥和代理地址
    - 步骤7：保存配置，重启 Claude Code
  - **国内使用方法二：配置代理**（包含环境变量设置）：
    - 步骤1：设置代理环境变量
      ```bash
      export https_proxy=http://127.0.0.1:7890
      export http_proxy=http://127.0.0.1:7890
      ```
    - 步骤2：设置 API 密钥
      ```bash
      export ANTHROPIC_API_KEY=sk-ant-xxxxx
      ```
    - 步骤3：验证连接
      ```bash
      claude -p "hello"
      # 预期输出：Hello! How can I help you today?
      ```
  - **首次启动和初始化**（包含每个操作步骤）：
    - 步骤1：在项目目录中运行 `claude`
    - 步骤2：Claude Code 会询问是否信任此项目，输入 `yes`
    - 步骤3：Claude Code 会扫描项目文件，等待完成
    - 步骤4：看到提示符 `>` 表示启动成功
  - **常见问题排查**（包含具体错误信息和解决方案）：
    - 问题1：`npm ERR! permission denied`
      - 原因：权限不足
      - 解决方案：使用 `sudo npm install -g @anthropic-ai/claude-code`
    - 问题2：`Error: Invalid API Key`
      - 原因：API 密钥格式错误或未设置
      - 解决方案：检查密钥格式，确保以 `sk-ant-` 开头
    - 问题3：`Network request failed`
      - 原因：网络连接问题
      - 解决方案：配置代理，检查防火墙设置
