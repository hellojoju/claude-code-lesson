## 技术栈

| 组件 | 选择 | 理由 |
|------|------|------|
| 内容格式 | Markdown + Frontmatter | 开发者友好，版本控制 |
| 构建工具 | 自定义 Node.js 脚本 | 灵活，可控 |
| 样式框架 | Tailwind CSS | 快速开发，响应式 |
| 代码高亮 | Prism.js / Shiki | 主流，支持多语言 |
| 搜索 | FlexSearch / Pagefind | 纯前端，无需后端 |
| 部署 | GitHub Pages / Vercel | 免费，自动部署 |
