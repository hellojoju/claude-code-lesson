# AI + 期货量化交易教程 设计规格

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。

**目标：** 为懂 AI 但不懂期货量化交易的用户，创建一套从零到专业的完整教程网站。

**架构：** 四阶段渐进式学习路径，每阶段有明确的学习目标和产出。

**技术栈：** 复用 Claude Code 中文教程网站的构建系统（Node.js + Markdown → HTML）

---

## 一、项目背景

### 1.1 目标用户画像

| 特征 | 描述 |
|------|------|
| AI 能力 | 熟练使用 AI 工具，能用 AI 生成代码 |
| 编程能力 | Python 入门级，不打算手写代码，用 AI 生成 |
| 期货知识 | 零基础，完全小白 |
| 量化知识 | 零基础 |
| 目标 | 自己学会 + 整理成可分享课程 |

### 1.2 核心设计原则

1. **喂到嘴边**：每个操作都讲清楚去哪、怎么找、怎么做
2. **从零开始**：假设读者完全不懂，解释每个概念
3. **AI 贯穿全程**：用 AI 学习、用 AI 写代码、用 AI 分析
4. **实战导向**：每章都有可运行的产出
5. **常识先行**：期货市场基础知识必须讲透

### 1.3 技术实现

复用 Claude Code 中文教程的构建系统：


## 三、阶段一详细设计（入门篇，13章）

### 01. 期货是什么？

**学习目标：** 理解期货的本质和存在的意义

**内容大纲：**
- 期货的起源故事（农产品远期合同）
- 为什么世界需要期货？
- 期货的两大功能：价格发现、风险管理
- 期货与股票的核心区别
  - 有到期日 vs 无到期日
  - 可以做空 vs 做空困难
  - 杠杆交易 vs 全额资金
  - T+0 vs T+1
- 你作为散户的角色定位

**实战练习：**
- 用 AI 解释"期货"和"股票"的区别
- 讨论：为什么有人选择期货而不是股票？

---

### 02. 期货市场的参与者

**学习目标：** 知道市场里有哪些人，你的对手是谁

**内容大纲：**
- 四类市场参与者
  - 套期保值者：企业，锁定成本/利润
  - 投机者：你，赚取价差
  - 套利者：赚取价差的价差
  - 做市商：提供流动性
- 套保者的案例
  - 钢厂买入铁矿石期货对冲涨价风险
  - 农户卖出玉米期货对冲跌价风险
- 你在市场中的角色
- 散户能赚钱吗？现实情况
- 量化能改变什么？

**实战练习：**
- 查看某品种的持仓龙虎榜
- 分析：谁是多头，谁是空头？

---

### 03. 套期保值：企业为什么做期货

**学习目标：** 理解企业参与期货的真正目的

**内容大纲：**
- 什么是套期保值？
- 具体案例：螺纹钢贸易商
  - 手里有 1000 吨现货
  - 担心价格下跌
  - 在期货市场卖出对冲
  - 结果：利润锁定
- 套保 vs 投机的区别
- 为什么这跟你有关系？
  - 套保盘影响市场价格
  - 理解他们，理解市场

**实战练习：**
- 找一份企业套期保值的新闻
- 用 AI 分析企业的套保逻辑

---

### 04. 期货交易的核心机制

**学习目标：** 理解保证金、杠杆、结算、爆仓

**内容大纲：**
- 保证金制度
  - 为什么能以小博大？
  - 保证金比例的含义
  - 螺纹钢 10% 保证金计算
- 杠杆的双刃剑
  - 涨 10% = 收益 100%
  - 跌 10% = 亏损 100%
- 每日结算制度
  - 盈亏每日划转
- 追加保证金 & 强制平仓
  - 什么情况会被要求补钱？
  - 不补会怎样？
- 什么情况会爆仓？
  - 仓位过重 + 方向错误
  - 如何避免：仓位控制、止损

**实战练习：**
- 用模拟账户体验爆仓过程
- 计算：不同杠杆下的盈亏情况

---

### 05. 合约、主力合约、换月

**学习目标：** 理解交易的是哪个合约

**内容大纲：**
- 期货合约的命名
  - 螺纹钢2505 = 2025年5月到期
- 为什么有多个合约？
- 主力合约是什么？
  - 成交量最大的合约
  - 行情软件默认显示
- 为什么要换月？
  - 合约快到期
  - 不换可能被交割
- 换月注意事项
  - 流动性变化
  - 价差变化

**实战练习：**
- 查看螺纹钢各合约成交量
- 找出当前主力合约

---

### 06. 交割是什么？

**学习目标：** 理解交割机制，散户如何避免

**内容大纲：**
- 交割 = 到期时一手交钱一手交货
- 谁参与交割？
  - 主要是企业
  - 散户一般不参与
- 散户如何避免交割？
  - 在最后交易日前平仓
  - 或换到下一个合约
- 忘了平仓会怎样？
  - 可能被强制交割
- 各品种的最后交易日

**实战练习：**
- 查询一个品种的交割规则

---

### 07. 交易成本：手续费、滑点、保证金

**学习目标：** 理解真实交易成本

**内容大纲：**
- 手续费
  - 开仓、平仓都要交
  - 不同品种费率不同
- 滑点
  - 什么是滑点？
  - 为什么会有滑点？
- 保证金计算
  - 公式：价格 × 交易单位 × 保证金比例
  - 实例计算
- 回测时必须考虑成本

**实战练习：**
- 计算做 1 手螺纹钢需要的资金
- 计算一次往返交易的成本

---

### 08. 交易时间与夜盘

**学习目标：** 理解交易时间安排

**内容大纲：**
- 日盘时间
  - 上午：9:00-11:30
  - 下午：13:30-15:00
- 夜盘时间
  - 各品种不同
  - 有色金属到次日 01:00
  - 贵金属到次日 02:30
  - 螺纹钢到 23:00
- 夜盘是必须的吗？
- 外盘影响
  - 铜看 LME
  - 黄金看 COMEX
  - 开盘可能跳空

**实战练习：**
- 查看有夜盘品种的分时图

---

### 09. 期货品种全景图

**学习目标：** 认识所有期货品种及其特点

**内容大纲：**
- 上海期货交易所（上期所）
  - 黑色系：螺纹钢、热卷
  - 有色金属：铜、铝、锌、镍
  - 贵金属：黄金、白银
  - 能源化工：原油、燃料油、橡胶
- 大连商品交易所（大商所）
  - 铁矿石、煤焦
  - 油脂油料：豆粕、豆油
  - 玉米、鸡蛋
  - 化工：塑料、PVC
- 郑州商品交易所（郑商所）
  - 软商品：棉花、白糖
  - 果品：苹果、红枣
  - 化工：PTA、甲醇
- 中国金融期货交易所（中金所）
  - 股指期货：IF、IH、IC、IM
  - 国债期货
- 新手应该从哪个品种开始？
  - 推荐：螺纹钢、豆粕、玉米
  - 不建议：原油、黄金、股指

**实战练习：**
- 下载行情软件
- 浏览所有品种
