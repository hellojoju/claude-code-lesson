文件系统集成

来源说明：本文内容基于 MCP 官方文档整理。

可靠性评级：A 官方文档

文件系统集成概述

MCP 文件系统服务器允许 Claude Code 安全地访问和操作文件系统。

功能特性

  • 读取文件内容
  • 写入和编辑文件
  • 浏览目录结构
  • 搜索文件内容
  • 文件监控

配置文件系统服务器

基本配置

{
  "servers": {
    "filesystem": {
      "command": "mcp-server-filesystem",
      "args": ["--root", "/path/to/project"]
    }
  }
}

多目录配置

{
  "servers": {
    "filesystem": {
      "command": "mcp-server-filesystem",
      "args": [
        "--root", "/path/to/project",
        "--allow", "/path/to/project/src",
        "--allow", "/path/to/project/tests",
        "--deny", "/path/to/project/.env"
      ]
    }
  }
}

权限配置

{
  "servers": {
    "filesystem": {
      "command": "mcp-server-filesystem",
      "permissions": {
        "read": true,
        "write": true,
        "delete": false,
        "allowedPaths": ["./src/**", "./tests/**"],
        "deniedPaths": ["./.env*", "./secrets/**"]
      }
    }
  }
}

文件监控

启用监控

{
  "servers": {
    "filesystem": {
      "watch": {
        "enabled": true,
        "paths": ["./src/**"],
        "ignore": ["./src/**/*.test.js"]
      }
    }
  }
}

监控事件

# 支持的监控事件
- add: 文件创建
- change: 文件修改
- unlink: 文件删除
- addDir: 目录创建
- unlinkDir: 目录删除

安全配置

路径限制

{
  "security": {
    "maxFileSize": 10485760,
    "allowedExtensions": [".js", ".ts", ".jsx", ".tsx", ".json", ".md"],
    "deniedExtensions": [".env", ".key", ".pem"],
    "sandbox": true
  }
}

敏感文件保护

{
  "deniedPaths": [
    "./.env",
    "./.env.*",
    "./**/secrets/**",
    "./**/*.key",
    "./**/*.pem",
    "./**/credentials.*"
  ]
}
