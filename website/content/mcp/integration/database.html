数据库集成

来源说明：本文内容基于 MCP 官方数据库服务器文档整理。

可靠性评级：A 官方文档

数据库集成概述

MCP 提供了多种数据库服务器的集成方案，让 Claude Code 能够直接查询和操作数据库。

支持的数据库

  • PostgreSQL
  • MySQL
  • SQLite
  • MongoDB
  • Redis

PostgreSQL 集成

安装服务器

# 使用 npm 安装
npm install -g @modelcontextprotocol/server-postgres

# 或使用 npx
npx @modelcontextprotocol/server-postgres

配置连接

// ~/.claude/mcp-config.json
{
  "servers": {
    "postgres": {
      "command": "mcp-server-postgres",
      "args": [],
      "env": {
        "DATABASE_URL": "postgresql://user:password@localhost:5432/mydb"
      }
    }
  }
}

使用示例

# 查询数据
> 查询 users 表的前 10 条记录

# Claude 会自动调用 MCP 工具
> /mcp call postgres.query --sql "SELECT * FROM users LIMIT 10"

# 分析表结构
> 分析 orders 表的结构

# 创建查询
> 写一个查询找出最近一周的订单

MySQL 集成

配置连接

{
  "servers": {
    "mysql": {
      "command": "mcp-server-mysql",
      "env": {
        "MYSQL_HOST": "localhost",
        "MYSQL_PORT": "3306",
        "MYSQL_USER": "root",
        "MYSQL_PASSWORD": "password",
        "MYSQL_DATABASE": "mydb"
      }
    }
  }
}

SQLite 集成

配置连接

{
  "servers": {
    "sqlite": {
      "command": "mcp-server-sqlite",
      "args": ["--db-path", "/path/to/database.db"]
    }
  }
}

使用示例

# 查询 SQLite 数据库
> 查询 database.db 中的所有表

# 执行查询
> SELECT name FROM sqlite_master WHERE type='table'

MongoDB 集成

配置连接

{
  "servers": {
    "mongodb": {
      "command": "mcp-server-mongodb",
      "env": {
        "MONGODB_URI": "mongodb://localhost:27017/mydb"
      }
    }
  }
}

使用示例

# 查询 MongoDB
> 查找 users 集合中年龄大于 18 的文档

# 聚合查询
> 统计每个城市的用户数量

安全配置

权限控制

{
  "servers": {
    "postgres": {
      "command": "mcp-server-postgres",
      "permissions": {
        "allow": ["SELECT", "INSERT", "UPDATE"],
        "deny": ["DELETE", "DROP", "TRUNCATE"],
        "tables": ["users", "orders", "products"],
        "maxRows": 1000
      }
    }
  }
}

连接池配置

{
  "servers": {
    "postgres": {
      "pool": {
        "min": 2,
        "max": 10,
        "idleTimeout": 30000
      }
    }
  }
}

最佳实践

查询优化

  • 使用 LIMIT 限制返回行数
  • 避免 SELECT *
  • 使用索引字段查询
  • 复杂查询分步执行

安全建议

  • 使用只读账户连接
  • 限制可访问的表
  • 启用查询审计
  • 定期检查权限配置
