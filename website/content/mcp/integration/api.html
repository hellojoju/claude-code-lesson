API集成

来源说明：本文内容基于 MCP 官方文档和社区实践整理。

可靠性评级：A 官方文档 + 社区验证

API集成概述

MCP 可以集成各种外部 API，让 Claude Code 能够调用 REST API、GraphQL 等服务。

支持的 API 类型

  • REST API
  • GraphQL
  • gRPC
  • WebSocket API

REST API 集成

配置 REST API 服务器

{
  "servers": {
    "my-api": {
      "command": "mcp-server-rest",
      "env": {
        "API_BASE_URL": "https://api.example.com",
        "API_KEY": "your-api-key"
      },
      "config": {
        "endpoints": [
          {
            "name": "getUsers",
            "method": "GET",
            "path": "/users",
            "description": "获取用户列表"
          },
          {
            "name": "getUser",
            "method": "GET",
            "path": "/users/{id}",
            "description": "获取单个用户"
          },
          {
            "name": "createUser",
            "method": "POST",
            "path": "/users",
            "description": "创建用户"
          }
        ]
      }
    }
  }
}

> 调用 getUsers 获取用户列表

# 带参数调用
> 调用 getUser，ID 为 123

GraphQL 集成

配置 GraphQL 服务器

{
  "servers": {
    "graphql-api": {
      "command": "mcp-server-graphql",
      "env": {
        "GRAPHQL_ENDPOINT": "https://api.example.com/graphql",
        "AUTH_TOKEN": "your-token"
      }
    }
  }
}

使用示例

# 执行 GraphQL 查询
> 查询所有用户的名称和邮箱

# Claude 会生成并执行 GraphQL 查询
query {
  users {
    name
    email
  }
}

# 执行变更
> 创建一个新用户

OpenAPI 集成

从 OpenAPI 规范生成

{
  "servers": {
    "petstore": {
      "command": "mcp-server-openapi",
      "args": ["--spec", "./openapi.yaml"],
      "env": {
        "API_KEY": "your-api-key"
      }
    }
  }
}

OpenAPI 规范示例

openapi: 3.0.0
info:
  title: Pet Store API
  version: 1.0.0
servers:
  - url: https://petstore.example.com/api
paths:
  /pets:
    get:
      summary: List all pets
      operationId: listPets
      responses:
        '200':
          description: A list of pets
    post:
      summary: Create a pet
      operationId: createPet
      requestBody:
        content:
          application/json:
            schema:
              $ref: '#/components/schemas/Pet'

认证配置

API Key 认证

{
  "headers": {
    "X-API-Key": "${API_KEY}"
  }
}

Bearer Token 认证

{
  "headers": {
    "Authorization": "Bearer ${AUTH_TOKEN}"
  }
}

OAuth 2.0 认证

{
  "auth": {
    "type": "oauth2",
    "clientId": "your-client-id",
    "clientSecret": "${CLIENT_SECRET}",
    "tokenUrl": "https://auth.example.com/oauth/token",
    "scopes": ["read", "write"]
  }
}

错误处理

错误重试配置

{
  "retry": {
    "maxAttempts": 3,
    "delay": 1000,
    "backoff": "exponential",
    "retryOn": [429, 500, 502, 503]
  }
}
