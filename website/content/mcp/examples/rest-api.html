REST API集成案例

来源说明：本文展示一个完整的 REST API MCP 集成案例。

可靠性评级：B 社区实践

案例背景

本案例演示如何集成一个电商 REST API，实现商品查询、订单管理等功能。

API 规范

基础 URL: https://api.shop.example.com/v1

端点:
- GET    /products      获取商品列表
- GET    /products/{id} 获取单个商品
- POST   /orders        创建订单
- GET    /orders/{id}   获取订单详情
- PUT    /orders/{id}   更新订单状态

配置 MCP

配置文件

// ~/.claude/mcp-config.json
{
  "servers": {
    "shop-api": {
      "command": "mcp-server-rest",
      "env": {
        "API_BASE_URL": "https://api.shop.example.com/v1",
        "API_KEY": "your-api-key-here"
      },
      "config": {
        "headers": {
          "Authorization": "Bearer ${API_KEY}",
          "Content-Type": "application/json"
        },
        "endpoints": [
          {
            "name": "listProducts",
            "method": "GET",
            "path": "/products",
            "description": "获取商品列表",
            "parameters": [
              {"name": "category", "in": "query", "type": "string"},
              {"name": "page", "in": "query", "type": "integer"},
              {"name": "limit", "in": "query", "type": "integer"}
            ]
          },
          {
            "name": "getProduct",
            "method": "GET",
            "path": "/products/{id}",
            "description": "获取商品详情",
            "parameters": [
              {"name": "id", "in": "path", "required": true, "type": "integer"}
            ]
          },
          {
            "name": "createOrder",
            "method": "POST",
            "path": "/orders",
            "description": "创建订单",
            "body": {
              "type": "object",
              "properties": {
                "product_id": {"type": "integer"},
                "quantity": {"type": "integer"},
                "user_id": {"type": "integer"}
              }
            }
          },
          {
            "name": "getOrder",
            "method": "GET",
            "path": "/orders/{id}",
            "description": "获取订单详情"
          },
          {
            "name": "updateOrder",
            "method": "PUT",
            "path": "/orders/{id}",
            "description": "更新订单状态"
          }
        ]
      }
    }
  }
}

使用示例

查询商品

# 查询商品列表
> 获取所有商品

# Claude 调用
GET /products

# 带参数查询
> 获取电子产品类别的商品，第 2 页

# Claude 调用
GET /products?category=electronics&page=2&limit=20

# 查询单个商品
> 获取商品 ID 为 123 的详情

# Claude 调用
GET /products/123

创建订单

# 创建订单
> 为用户 1 创建一个订单，商品 ID 123，数量 2

# Claude 调用
POST /orders
{
  "user_id": 1,
  "product_id": 123,
  "quantity": 2
}

更新订单

# 更新订单状态
> 将订单 456 的状态更新为已发货

# Claude 调用
PUT /orders/456
{
  "status": "shipped"
}

高级用法

错误处理

{
  "errorHandling": {
    "retry": {
      "maxAttempts": 3,
      "delay": 1000,
      "retryOn": [429, 500, 502, 503]
    },
    "fallback": {
      "enabled": true,
      "message": "API 暂时不可用，请稍后重试"
    }
  }
}

缓存配置

{
  "cache": {
    "enabled": true,
    "ttl": 300,
    "methods": ["GET"],
    "excludePaths": ["/orders"]
  }
}

请求限流

{
  "rateLimit": {
    "enabled": true,
    "requests": 100,
    "window": 60000
  }
}
