系统命令详解

来源说明：本文内容基于 Claude Code 官方文档和实际使用案例整理。

可靠性评级：A 官方文档 + 实践验证

/run - 运行命令

基本用法

# 运行 npm 命令
> /run npm install

# 运行构建命令
> /run npm run build

# 运行测试
> /run npm test

高级选项

# 指定工作目录
> /run npm install --cwd ./frontend

# 设置超时
> /run npm install --timeout 300000

# 环境变量
> /run npm run deploy --env NODE_ENV=production

# 后台运行
> /run npm start --background

实际案例

# 安装依赖并检查
> /run npm install
> /run npm audit

# 运行多个命令
> /run npm run lint && npm run test

# 使用 pnpm 或 yarn
> /run pnpm install
> /run yarn build

/shell - Shell 命令执行

安全提示：Shell 命令具有系统级权限，请确保在安全环境中使用。

基本用法

# 执行单个命令
> /shell ls -la

# 查看文件内容
> /shell cat package.json

# 查看进程
> /shell ps aux | grep node

管道和重定向

# 管道操作
> /shell cat file.txt | grep "pattern"

# 输出重定向
> /shell echo "content" > file.txt

# 追加内容
> /shell echo "more" >> file.txt

# 错误重定向
> /shell command 2>&1 | tee output.log

实际案例

# 查看 Git 状态
> /shell git status

# 查看 Git 日志
> /shell git log --oneline -10

# 查看磁盘使用
> /shell df -h

# 查看内存使用
> /shell free -m

# 查找大文件
> /shell find . -type f -size +100M

/install - 安装依赖

基本用法

# 安装 npm 包
> /install lodash

# 安装开发依赖
> /install eslint --save-dev

# 安装特定版本
> /install react@18.2.0

# 全局安装
> /install typescript --global

批量安装

# 从 package.json 安装
> /install

# 安装多个包
> /install react react-dom @types/react

# 安装特定依赖组
> /install --production

其他包管理器

# 使用 yarn
> /install lodash --manager yarn

# 使用 pnpm
> /install lodash --manager pnpm

# 使用 pip (Python)
> /install requests --manager pip

Git 相关命令

状态查看

# 查看状态
> /git status

# 查看差异
> /git diff

# 查看日志
> /git log --oneline -10

# 查看分支
> /git branch -a

提交操作

# 添加文件
> /git add .

# 提交
> /git commit -m "feat: add new feature"

# 推送
> /git push origin main

分支操作

# 创建分支
> /git checkout -b feature/new-feature

# 切换分支
> /git checkout main

# 合并分支
> /git merge feature/new-feature

Docker 命令

容器操作

# 查看容器
> /shell docker ps -a

# 构建镜像
> /shell docker build -t my-app .

# 运行容器
> /shell docker run -p 3000:3000 my-app

# 查看日志
> /shell docker logs container-name

镜像操作

# 查看镜像
> /shell docker images

# 拉取镜像
> /shell docker pull node:18

# 删除镜像
> /shell docker rmi image-id

进程管理

查看进程

# 查看所有进程
> /shell ps aux

# 查看特定进程
> /shell ps aux | grep node

# 查看端口占用
> /shell lsof -i :3000

# 查看资源使用
> /shell top -n 1

终止进程

# 终止进程
> /shell kill PID

# 强制终止
> /shell kill -9 PID

# 终止所有 node 进程
> /shell pkill node

网络命令

网络诊断

# 测试连接
> /shell ping google.com

# 查看端口
> /shell netstat -an | grep LISTEN

# DNS 查询
> /shell nslookup google.com

# HTTP 请求
> /shell curl -I https://api.example.com

下载文件

# 使用 curl
> /shell curl -O https://example.com/file.zip

# 使用 wget
> /shell wget https://example.com/file.zip
