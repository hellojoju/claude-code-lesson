斜杠命令介绍

来源说明：本文内容基于 Claude Code 官方文档和社区最佳实践整理。

可靠性评级：A 官方文档

什么是斜杠命令

斜杠命令（Slash Commands）是 Claude Code 的核心交互方式，通过以 / 开头的指令来触发特定功能。这种设计让用户可以精确控制 Claude 的行为，而不是仅依赖自然语言理解。

命令的优势

  • 精确性：明确指定要执行的操作
  • 效率：快速调用常用功能
  • 可扩展：支持自定义命令
  • 可组合：多个命令可以组合使用

命令语法

基本语法

/命令名 [参数] [选项]

语法示例

# 无参数命令
/help

# 带参数命令
/read src/index.js

# 带选项命令
/search "function" --type js

# 多参数命令
/write test.txt "Hello World"

命令分类

分类 说明 常用命令
文件操作 读写、搜索文件 /read, /write, /search
系统命令 执行系统操作 /run, /shell, /exec
会话管理 管理对话会话 /clear, /save, /load
配置管理 查看和修改配置 /config, /set
调试工具 调试和诊断 /debug, /log, /inspect
自定义命令 用户定义的命令 /my-command

内置命令一览

帮助和信息

/help          显示帮助信息
/version       显示版本信息
/status        显示当前状态

文件操作

/read          读取文件内容
/write         写入文件
/edit          编辑文件
/search        搜索文件内容
/glob          按模式查找文件

代码操作

/analyze       分析代码
/refactor      重构代码
/test          运行测试
/lint          代码检查

系统操作

/run           运行命令
/shell         执行 shell 命令
/install       安装依赖

会话管理

/clear         清除对话
/save          保存会话
/load          加载会话
/export        导出对话

基本使用方法

查看帮助

# 查看所有命令
> /help

# 查看特定命令帮助
> /help read

# 查看命令详细用法
> /read --help

自动补全

Claude Code 支持命令自动补全：

输入 /re 然后按 Tab 键
→ 自动补全为 /read

命令历史

按 ↑ 键浏览历史命令
按 Ctrl+R 搜索历史命令

使用技巧

命令组合

# 先读取文件，再分析
> /read package.json
> 分析这个项目的依赖关系

# 搜索后修改
> /search "TODO" --type js
> 把所有 TODO 改成 FIXME

管道操作

# 搜索结果作为输入
> /search "function" --type js | 分析这些函数的复杂度

批量操作

# 批量处理文件
> /glob "*.js" | 给所有文件添加版权声明

自定义命令简介

Claude Code 支持创建自定义命令来扩展功能：

自定义命令位置

~/.claude/commands/          # 用户级命令
.project/commands/           # 项目级命令

简单自定义命令示例

# ~/.claude/commands/review.md

你是一个代码审查专家。请审查当前文件或选中的代码，关注：
1. 代码质量
2. 潜在bug
3. 性能问题
4. 安全隐患
5. 改进建议

使用自定义命令：

> /review src/index.js
