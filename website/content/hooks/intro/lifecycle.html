生命周期钩子

来源说明：本文内容基于 Claude Code 官方文档整理。

可靠性评级：A 官方文档

生命周期概述

生命周期钩子在 Claude Code 的关键生命周期事件触发，允许在特定阶段执行自定义逻辑。

生命周期阶段

启动 → 初始化 → 运行 → 关闭

每个阶段都有对应的钩子：
- onStartup: 启动时
- onInit: 初始化时
- onReady: 准备就绪
- onShutdown: 关闭时

启动钩子 (onStartup)

触发时机

Claude Code 进程启动时，在任何初始化之前触发。

使用场景

  • 检查环境依赖
  • 加载全局配置
  • 初始化日志系统

示例

// hooks/startup.js
module.exports = async (context) => {
  console.log('Claude Code 正在启动...');
  
  // 检查 Node.js 版本
  const nodeVersion = process.version;
  if (nodeVersion < 'v18.0.0') {
    return {
      action: 'error',
      message: '需要 Node.js 18 或更高版本'
    };
  }
  
  // 加载环境变量
  require('dotenv').config();
  
  return { action: 'continue' };
};

初始化钩子 (onInit)

触发时机

Claude Code 初始化时，在加载配置和插件之后触发。

使用场景

  • 初始化数据库连接
  • 注册自定义命令
  • 设置事件监听器

示例

// hooks/init.js
module.exports = async (context) => {
  const { config, registerCommand } = context;
  
  // 初始化数据库连接
  const db = await connectDatabase(config.database);
  context.db = db;
  
  // 注册自定义命令
  registerCommand('my-command', async () => {
    // 命令逻辑
  });
  
  return { action: 'continue' };
};

就绪钩子 (onReady)

触发时机

Claude Code 完全初始化并准备接受用户输入时触发。

使用场景

  • 显示欢迎信息
  • 启动后台服务
  • 执行启动任务

示例

// hooks/ready.js
module.exports = async (context) => {
  // 显示欢迎信息
  console.log('欢迎使用 Claude Code！');
  console.log('版本:', context.version);
  
  // 启动文件监控
  if (context.config.watch) {
    startFileWatcher();
  }
  
  // 检查更新
  checkForUpdates();
  
  return { action: 'continue' };
};

关闭钩子 (onShutdown)

触发时机

Claude Code 关闭时触发，用于清理资源。

使用场景

  • 关闭数据库连接
  • 保存未保存的数据
  • 清理临时文件

示例

// hooks/shutdown.js
module.exports = async (context) => {
  console.log('正在关闭 Claude Code...');
  
  // 保存会话状态
  await saveSessionState(context.session);
  
  // 关闭数据库连接
  if (context.db) {
    await context.db.close();
  }
  
  // 清理临时文件
  cleanupTempFiles();
  
  console.log('再见！');
  return { action: 'continue' };
};

会话钩子

会话开始 (onSessionStart)

module.exports = async (context) => {
  console.log('新会话开始');
  
  // 加载项目记忆
  await loadProjectMemory();
  
  // 恢复上次会话状态
  await restoreLastSession();
  
  return { action: 'continue' };
};

会话结束 (onSessionEnd)

module.exports = async (context) => {
  console.log('会话结束');
  
  // 保存会话数据
  await saveSessionData(context.session);
  
  // 更新项目记忆
  await updateProjectMemory();
  
  return { action: 'continue' };
};

配置生命周期钩子

// hooks.json
{
  "lifecycle": {
    "onStartup": {
      "handlers": [
        { "script": "hooks/startup.js", "priority": 100 }
      ]
    },
    "onInit": {
      "handlers": [
        { "script": "hooks/init.js", "priority": 50 }
      ]
    },
    "onReady": {
      "handlers": [
        { "script": "hooks/ready.js", "priority": 10 }
      ]
    },
    "onShutdown": {
      "handlers": [
        { "script": "hooks/shutdown.js", "priority": 100 }
      ]
    }
  }
}
