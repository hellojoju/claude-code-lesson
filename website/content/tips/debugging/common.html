常见问题排查

可靠性: A 来源: Claude Code 官方文档 + 社区经验

常见错误及解决方案

安装问题

问题：npm 安装失败

# 解决方案：使用官方安装脚本
curl -fsSL https://claude.ai/install.sh | sh

# 或使用 npx
npx @anthropic-ai/claude-code

问题：权限错误

# 解决方案：修复 npm 权限
sudo chown -R $(whoami) ~/.npm
sudo chown -R $(whoami) /usr/local/lib/node_modules

认证问题

问题：API Key 无效

# 检查 API Key 配置
cat ~/.claude/config.json

# 重新设置 API Key
claude config set apiKey YOUR_API_KEY

# 或使用环境变量
export ANTHROPIC_API_KEY=YOUR_API_KEY

问题：认证过期

# 重新登录
claude logout
claude login

文件操作问题

问题：文件访问被拒绝

// 检查权限配置
// ~/.claude/settings.json
{
  "permissions": {
    "allow": ["read(**)", "write(src/**)"]
  }
}

问题：找不到文件

# 检查 .claudeignore 配置
cat .claudeignore

# 使用绝对路径
claude "读取 /absolute/path/to/file.ts"

性能问题

问题：响应缓慢

  • 使用 /compact 压缩历史
  • 减少上下文文件数量
  • 检查网络连接
  • 使用更快的模型

问题：内存占用高

# 清理缓存
rm -rf ~/.claude/cache/*

# 重启 Claude Code
claude --restart

调试技巧

# 启用调试模式
export CLAUDE_DEBUG=1
export CLAUDE_LOG_LEVEL=debug

# 查看日志
tail -f ~/.claude/logs/claude-$(date +%Y-%m-%d).log

# 详细输出
claude --verbose "your prompt"
