隐藏功能

可靠性: A 来源: Claude Code 官方文档 + 社区发现

发现更多功能

Claude Code 有一些不为人知但非常实用的功能，本节将为你揭示这些隐藏的宝藏。

环境变量

变量 功能
CLAUDE_DEBUG 启用调试模式
CLAUDE_LOG_LEVEL 设置日志级别
CLAUDE_VERBOSE 详细输出模式
CLAUDE_PERF_MONITOR 性能监控
ANTHROPIC_API_KEY API 密钥

隐藏命令

诊断命令

# 查看系统信息
claude --version
claude --info

# 诊断模式
claude --diagnose

# 检查配置
claude config check

会话管理

# 恢复上次会话
claude --resume

# 列出历史会话
claude --sessions

# 加载特定会话
claude --session SESSION_ID

高级配置

自定义提示词模板

// ~/.claude/templates.json
{
  "review": "作为代码审查专家，审查以下代码：\n{code}",
  "test": "为以下代码生成单元测试：\n{code}",
  "doc": "为以下代码生成文档：\n{code}"
}

自定义命令别名

// ~/.claude/aliases.json
{
  "r": "/review",
  "t": "/test",
  "d": "/doc",
  "c": "/compact"
}

实用技巧

多行输入

使用 """ 包裹多行内容：

"""
分析以下代码：
function hello() {
    console.log('Hello');
}
"""

文件引用

# 使用 @ 引用文件
"审查 @src/api/users.ts 的代码"

# 使用 # 引用行号
"修改 @src/index.ts #10-20"

上下文控制

# 只读模式
claude --read-only

# 限制文件范围
claude --scope src/

# 禁用记忆
claude --no-memory

实验性功能

# 启用实验性功能
export CLAUDE_EXPERIMENTAL=1

# 可用的实验性功能：
# - 多模型协作
# - 自动代码补全
# - 实时协作模式
