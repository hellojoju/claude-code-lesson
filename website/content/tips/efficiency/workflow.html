工作流技巧

可靠性: A 来源: Claude Code 官方文档 + 社区经验

优化工作流程

通过合理配置和使用技巧，可以显著提升 Claude Code 的使用效率。

快捷键和命令

命令 功能
/help 显示帮助信息
/clear 清除对话历史
/compact 压缩对话历史
/config 查看/修改配置
/memory 管理记忆
Ctrl+C 中断当前操作

效率提升技巧

1. 使用 CLAUDE.md

在项目根目录创建 CLAUDE.md 文件，提供项目上下文：

# 项目上下文

## 技术栈
- React + TypeScript
- Tailwind CSS
- Prisma + PostgreSQL

## 编码规范
- 使用函数式组件
- 遵循 ESLint 配置
- 组件命名 PascalCase

2. 配置 .claudeignore

# 排除不需要的文件
node_modules/
dist/
*.min.js
.env*

3. 利用记忆功能

# 保存重要信息到记忆
"记住：项目使用 PostgreSQL 数据库，连接字符串在 .env 中"

# 查看记忆
/memory list

4. 创建常用 Skills

# skills/code-review/skill.yaml
id: code-review
name: Code Review
triggers:
  - type: command
    value: /review

常见工作流

功能开发工作流

1. /compact          # 清理上下文
2. 描述需求           # 详细说明功能
3. "创建功能分支"     # Git 操作
4. 编写代码           # AI 辅助
5. "运行测试"         # 验证功能
6. "创建 PR"          # 提交审查

问题排查工作流

1. 描述问题现象
2. 提供错误信息
3. "分析可能的原因"
4. "提供解决方案"
5. "实施修复"
6. "验证修复结果"
