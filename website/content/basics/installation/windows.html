Windows 安装详解

可靠性: A 来源: Claude Code 官方文档

预计阅读时间: 25 分钟 | 难度: 初级

安装前准备

在 Windows 上安装 Claude Code，您需要先安装 WSL 2（Windows Subsystem for Linux）。

重要：Claude Code 目前不支持直接在 Windows 上运行，必须通过 WSL 2 使用。

系统要求

  • Windows 10 版本 2004+ 或 Windows 11
  • 支持虚拟化的 CPU
  • 至少 8GB RAM（推荐 16GB）
  • 至少 20GB 可用磁盘空间

安装 WSL 2

步骤 1：启用 WSL

以管理员身份打开 PowerShell，运行：

wsl --install

预期输出：

正在安装: Windows 子系统 for Linux
Windows 子系统 for Linux 已安装。
正在安装: Ubuntu
Ubuntu 已安装。
来源： Microsoft WSL 官方文档

步骤 2：重启计算机

安装完成后，重启计算机以完成 WSL 2 的安装。

步骤 3：设置 Ubuntu

重启后，Ubuntu 会自动启动。首次启动需要设置用户名和密码：

Enter new UNIX username: your_username
New password: 
Retype new password:

安装 Node.js

步骤 1：更新包管理器

在 Ubuntu 终端中运行：

sudo apt update && sudo apt upgrade -y

步骤 2：安装 Node.js 20.x

curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

验证安装：

node --version
# 预期输出：v20.x.x
npm --version
# 预期输出：10.x.x
来源： NodeSource GitHub 仓库

安装 Claude Code

步骤 1：安装 Claude Code

npm install -g @anthropic-ai/claude-code

预期输出：

added 1 package in 5s

步骤 2：验证安装

claude --version
# 预期输出：claude-code version x.x.x

步骤 3：配置 API 密钥

export ANTHROPIC_API_KEY=sk-ant-xxxxx
获取 API 密钥： Anthropic Console

步骤 4：启动 Claude Code

claude

预期输出：

Welcome to Claude Code!
Version: x.x.x

Do you trust this project? (yes/no)

常见问题

问题 1：WSL 2 安装失败

解决方案：

# 启用虚拟机平台
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

# 启用 WSL
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

# 重启计算机
# 下载并安装 WSL 2 Linux 内核更新包
# https://aka.ms/wsl2kernel

问题 2：npm 权限错误

解决方案：

# 方法 1：使用 sudo
sudo npm install -g @anthropic-ai/claude-code

# 方法 2：修改 npm 全局目录
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
