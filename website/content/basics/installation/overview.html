什么是 Claude Code？

Claude Code 是 Anthropic 开发的系统级 AI Agent，它不仅是一个代码编写工具，更是一个可以通过自然语言指令完成各种电脑任务的智能助手。

代码编写

自动生成、编辑和优化代码

调试修复

自动发现和修复代码问题

文件管理

智能文件操作和项目管理

命令执行

安全执行系统命令和脚本

来源： Claude Code 官方页面

系统要求

操作系统 最低版本 推荐版本
macOS 10.15 (Catalina) 14.0+ (Sonoma)
Ubuntu/Debian 18.04 / 10 22.04 / 12
Windows WSL 2 WSL 2 with Ubuntu 22.04

软件依赖

  • Node.js: v18.0 或更高版本
  • npm: 随 Node.js 自动安装
  • Git: 用于版本控制（Windows 用户必需）
来源： Claude Code 官方仓库

快速安装

方法一：官方安装脚本（推荐）

curl -fsSL https://claude.ai/install.sh | bash
来源： Claude Code 官方页面

方法二：npm 安装

npm install -g @anthropic-ai/claude-code
来源： Claude Code 官方仓库
