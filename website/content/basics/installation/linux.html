Linux 安装详解

可靠性: A 来源: Claude Code 官方文档

更新时间: 2025-04-04 作者: Claude Code 教程团队

系统要求

最低配置

操作系统 Ubuntu 20.04+, Debian 11+, Fedora 35+, Arch Linux
Node.js v18.0.0 或更高版本
内存 4GB RAM（推荐 8GB+）
磁盘空间 500MB 可用空间

Ubuntu/Debian 安装

方法一：使用 NodeSource（推荐）

# 添加 Node.js 20.x 仓库
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# 安装 Node.js
sudo apt install -y nodejs

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

方法二：使用原生安装器

# 使用官方安装脚本
curl -fsSL https://claude.ai/install.sh | bash

提示

如果遇到权限问题，不要使用 sudo npm install，推荐使用原生安装器。

Fedora/RHEL/CentOS 安装

# Fedora
sudo dnf install -y nodejs npm

# RHEL/CentOS (需要启用 EPEL)
sudo dnf install -y epel-release
sudo dnf install -y nodejs npm

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

使用 NodeSource

# 添加 Node.js 20.x 仓库
curl -fsSL https://rpm.nodesource.com/setup_20.x | sudo bash -

# 安装 Node.js
sudo dnf install -y nodejs

Arch Linux 安装

# 使用 pacman 安装 Node.js
sudo pacman -S nodejs npm

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

使用 AUR

# 使用 yay 安装（如果可用）
yay -S claude-code

使用 nvm 安装

nvm (Node Version Manager) 允许你管理多个 Node.js 版本：

# 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新加载 shell 配置
source ~/.bashrc

# 安装 Node.js 20
nvm install 20

# 使用 Node.js 20
nvm use 20

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

验证安装

# 检查版本
claude --version

# 运行健康检查
claude doctor

# 首次启动
claude

首次登录

首次运行时，Claude Code 会引导你完成登录：

  1. 选择登录方式（浏览器或 API Key）
  2. 完成 Anthropic 账户授权
  3. 选择订阅计划（Pro 或 Max）

常见问题

权限错误？

不要使用 sudo npm install，改用原生安装器：

curl -fsSL https://claude.ai/install.sh | bash

Node.js 版本过低？

确保 Node.js 版本 >= 18：

node --version
# 如果版本过低，使用 nvm 安装新版本
nvm install 20
nvm use 20

命令找不到？

确保 npm 全局目录在 PATH 中：

# 添加到 ~/.bashrc 或 ~/.zshrc
export PATH="$(npm config get prefix)/bin:$PATH"

# 重新加载配置
source ~/.bashrc

WSL 问题？

在 WSL 中安装时，确保 Windows 上正确安装了 WSL：

# 在 Windows PowerShell 中运行
wsl --install

# 然后在 WSL 中安装 Claude Code
