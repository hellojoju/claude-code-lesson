快速开始

可靠性: A 来源: Claude Code 官方文档

更新时间: 2025-04-04 作者: Claude Code 教程团队

首次启动

安装完成后，在项目目录中启动 Claude Code：

# 进入项目目录
cd /path/to/your/project

# 启动 Claude Code
claude

看到提示符 > 表示启动成功。

提示：Claude Code 会自动识别当前目录的项目类型，并加载相关上下文。

启动选项

选项 说明
claude 启动新会话
claude -c 继续上次会话
claude --resume 恢复指定会话
claude -p "任务" 执行单次任务

登录认证

首次运行时，Claude Code 会引导你完成登录：

方式一：浏览器登录（推荐）

? Choose an authentication method:
  ❯ Continue with browser
    Continue with API key

选择浏览器登录后，会自动打开浏览器完成授权。

方式二：API Key 登录

# 从 console.anthropic.com 获取 API Key
? Enter your API key: sk-ant-...

基本命令

斜杠命令

> /help          # 显示帮助
> /clear         # 清除当前会话
> /compact       # 压缩上下文
> /context       # 查看上下文使用情况
> /cost          # 查看 token 成本
> /login         # 重新登录
> /logout        # 登出

文件操作

> /read file.js           # 读取文件
> /write test.txt "内容"  # 写入文件
> 查看 @package.json      # 使用 @ 语法引用文件

自然语言

# 你可以直接用自然语言与 Claude 交流
> 这个项目是做什么的？
> 帮我创建一个登录组件
> 修复这个错误：[粘贴错误信息]
> 解释一下这段代码的作用

第一个任务

让我们完成一个简单的任务来熟悉 Claude Code：

任务：理解项目结构

# 启动 Claude Code
cd your-project
claude

# 询问项目信息
> 这个项目是做什么的？使用什么技术栈？

# 查看主要文件
> 项目的主入口在哪里？

# 了解架构
> 项目的目录结构是怎样的？

任务：创建新功能

# 描述需求
> 创建一个用户登录表单组件，包含用户名和密码输入

# Claude 会：
# 1. 分析项目结构
# 2. 确定技术栈
# 3. 创建组件文件
# 4. 添加必要样式

新手技巧

具体描述

尽量具体地描述你的需求，避免模糊的指令。

使用 @ 引用

使用 @文件名 引用文件，让 Claude 更准确理解上下文。

审查确认

仔细审查 Claude 的修改建议，确保符合预期。

及时提交

重要修改前先提交 Git，方便回滚。

权限管理

默认情况下，Claude 在执行操作前会请求确认：

  • Enter - 允许执行
  • Esc - 拒绝执行
  • e - 编辑后执行

按 Shift+Tab 可以切换自动接受模式。
