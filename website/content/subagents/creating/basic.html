创建子代理

来源说明：本文内容基于 Claude Code 官方文档和社区最佳实践整理。

可靠性评级：A 官方文档 + 社区验证

快速开始

创建第一个子代理

# 创建子代理目录
mkdir -p ~/.claude/subagents/my-agent

# 创建配置文件
cat > ~/.claude/subagents/my-agent/agent.json << 'EOF'
{
  "name": "my-agent",
  "description": "我的自定义代理",
  "version": "1.0.0"
}
EOF

# 创建提示词文件
cat > ~/.claude/subagents/my-agent/prompt.md << 'EOF'
你是一个专门的代理，负责处理特定任务。

任务：{{task}}

请按照以下步骤执行：
1. 分析任务
2. 执行操作
3. 返回结果
EOF

配置文件详解

基础配置

{
  "name": "agent-name",
  "version": "1.0.0",
  "description": "代理描述",
  "author": "作者名"
}

完整配置

{
  "name": "code-reviewer",
  "version": "1.0.0",
  "description": "代码审查代理",
  "author": "your-name",
  
  "capabilities": [
    "code-review",
    "security-check",
    "style-check"
  ],
  
  "triggers": {
    "manual": true,
    "keywords": ["审查", "review", "检查代码"],
    "patterns": ["review\\s+this", "check\\s+code"]
  },
  
  "tools": [
    "Read",
    "SearchCodebase",
    "Grep"
  ],
  
  "prompt": "prompt.md",
  
  "context": {
    "maxTokens": 4000,
    "includeFiles": true,
    "includeMemory": true
  },
  
  "execution": {
    "timeout": 60000,
    "maxRetries": 3,
    "parallel": false
  },
  
  "output": {
    "format": "markdown",
    "stream": true
  }
}

提示词文件

基本结构

# prompt.md

你是一个代码审查专家代理。

## 任务
{{task}}

## 上下文
{{#if context.file}}
文件：{{context.file.path}}
内容：
```
{{context.file.content}}
```
{{/if}}

## 审查要求
1. 检查代码质量
2. 发现安全问题
3. 提供改进建议

## 输出格式
请按以下格式输出审查结果：
- 问题列表
- 严重程度
- 修复建议

使用变量

变量 说明
{{task}} 分配的任务描述
{{context}} 传递的上下文
{{params}} 任务参数
{{memory}} 相关记忆

工具配置

可用工具

{
  "tools": [
    "Read",           // 读取文件
    "Write",          // 写入文件
    "Edit",           // 编辑文件
    "SearchCodebase", // 搜索代码
    "Grep",           // 正则搜索
    "Glob",           // 文件匹配
    "LS",             // 列出目录
    "RunCommand",     // 执行命令
    "WebSearch",      // 网络搜索
    "WebFetch"        // 获取网页
  ]
}

工具权限控制

{
  "tools": {
    "Read": {
      "allowed": true,
      "paths": ["./**"]
    },
    "Write": {
      "allowed": true,
      "paths": ["./src/**"],
      "exclude": ["./**/.env*"]
    },
    "RunCommand": {
      "allowed": true,
      "commands": ["npm", "git", "node"]
    }
  }
}

触发器配置

手动触发

{
  "triggers": {
    "manual": true
  }
}

关键词触发

{
  "triggers": {
    "keywords": ["审查", "review", "检查"],
    "caseSensitive": false
  }
}

模式匹配触发

{
  "triggers": {
    "patterns": [
      "review\\s+(this\\s+)?code",
      "check\\s+(the\\s+)?(file|code)"
    ]
  }
}

条件触发

{
  "triggers": {
    "conditions": {
      "filePattern": "*.js",
      "hasSelection": true,
      "minSelectionLength": 10
    }
  }
}

测试子代理

验证配置

# 验证配置文件
> /subagent validate my-agent

# 检查提示词
> /subagent check-prompt my-agent

测试运行

# 测试运行
> /subagent test my-agent --task "审查代码"

# 调试模式
> /subagent test my-agent --debug

# 查看日志
> /subagent logs my-agent

完整示例

代码审查代理

// agent.json
{
  "name": "security-reviewer",
  "version": "1.0.0",
  "description": "安全审查代理",
  
  "capabilities": ["security-audit", "vulnerability-check"],
  
  "triggers": {
    "keywords": ["安全审查", "security review", "漏洞检查"]
  },
  
  "tools": ["Read", "Grep", "SearchCodebase"],
  
  "prompt": "prompt.md",
  
  "execution": {
    "timeout": 120000
  }
}
# prompt.md

你是一位安全专家代理，专门负责代码安全审查。

## 任务
{{task}}

## 代码
```
{{context.file.content}}
```

## 审查重点

### 1. 注入漏洞
- SQL 注入
- XSS 攻击
- 命令注入

### 2. 认证授权
- 弱密码
- 会话管理
- 权限检查

### 3. 敏感数据
- 密钥泄露
- 日志敏感信息
- 不安全存储

## 输出格式

### 🔴 高危漏洞
[列出高危问题]

### 🟡 中危漏洞
[列出中危问题]

### 🟢 低危问题
[列出低危问题]

### 修复建议
[提供具体修复方案]
