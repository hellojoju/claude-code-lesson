管理记忆详解

来源说明：本文内容基于 Claude Code 官方文档和实际使用案例整理。

可靠性评级：A 官方文档 + 实践验证

查看记忆

列出所有记忆

# 列出所有记忆
> /memory list

# 按类型列出
> /memory list --type project
> /memory list --type user
> /memory list --type global

# 按标签列出
> /memory list --tags "config"

# 分页显示
> /memory list --page 1 --page-size 20

输出格式

记忆列表
========

ID: mem-001
类型: project
标签: [config, api]
创建: 2026-04-04 10:00:00
内容: API 基础地址: https://api.example.com/v1

ID: mem-002
类型: user
标签: [preferences, style]
创建: 2026-04-04 10:30:00
内容: 代码风格: 单引号，无分号

共 2 条记忆

搜索记忆

关键词搜索

# 搜索关键词
> /memory search "API"

# 搜索多个关键词
> /memory search "React TypeScript"

# 正则搜索
> /memory search "/api\..*\.com/g"

高级搜索

# 按类型搜索
> /memory search "config" --type project

# 按标签搜索
> /memory search "style" --tags "preferences"

# 按日期范围搜索
> /memory search "" --from 2026-01-01 --to 2026-04-04

# 组合搜索
> /memory search "React" --type project --tags "config"

更新记忆

修改记忆内容

# 更新记忆内容
> /memory update mem-001 "新内容: API 基础地址已更新"

# 追加内容
> /memory append mem-001 "\n注意: 需要添加 API Key"

# 替换内容
> /memory replace mem-001 "旧地址" "新地址"

修改记忆属性

# 更新标签
> /memory tag mem-001 --add "important"
> /memory tag mem-001 --remove "temp"

# 更新优先级
> /memory priority mem-001 100

# 更新过期时间
> /memory expire mem-001 30d

删除记忆

删除单个记忆

# 按 ID 删除
> /memory delete mem-001

# 确认删除
> /memory delete mem-001 --confirm

批量删除

# 按类型删除
> /memory clear --type session

# 按标签删除
> /memory clear --tags "temp"

# 删除过期记忆
> /memory clear --expired

# 删除所有记忆（危险操作）
> /memory clear --all --confirm

警告：删除操作不可逆，请谨慎操作！

导出和导入

导出记忆

# 导出所有记忆
> /memory export memories.json

# 导出特定类型
> /memory export project-memories.json --type project

# 导出为不同格式
> /memory export memories.md --format markdown
> /memory export memories.yaml --format yaml

导入记忆

# 从文件导入
> /memory import memories.json

# 导入并覆盖
> /memory import memories.json --overwrite

# 导入并合并
> /memory import memories.json --merge

导出格式示例

{
  "version": "1.0",
  "exported": "2026-04-04T10:00:00Z",
  "memories": [
    {
      "id": "mem-001",
      "type": "project",
      "content": "API 基础地址: https://api.example.com/v1",
      "tags": ["config", "api"],
      "created": "2026-04-04T10:00:00Z",
      "priority": 10
    }
  ]
}

备份和恢复

创建备份

# 创建完整备份
> /memory backup create

# 创建带标签的备份
> /memory backup create --label "before-refactor"

# 查看备份列表
> /memory backup list

恢复备份

# 恢复最近备份
> /memory backup restore

# 恢复特定备份
> /memory backup restore backup-2026-04-04

# 恢复前确认
> /memory backup restore --dry-run
