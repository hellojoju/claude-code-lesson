文档自动化价值

文档是项目的重要组成部分，但往往被忽视。Claude Code 可以帮助自动生成和维护文档，确保文档与代码同步更新。

API 文档模板

# API 文档

## 用户接口

### 获取用户列表

\`\`\`
GET /api/v1/users
\`\`\`

**查询参数**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| page | number | 否 | 页码，默认 1 |
| limit | number | 否 | 每页数量，默认 20 |
| search | string | 否 | 搜索关键词 |

**响应示例**

\`\`\`json
{
  "code": 0,
  "data": {
    "users": [
      {
        "id": "user_123",
        "name": "张三",
        "email": "zhangsan@example.com"
      }
    ],
    "total": 100,
    "page": 1,
    "limit": 20
  }
}
\`\`\`

**错误码**

| 错误码 | 描述 |
|--------|------|
| 1001 | 参数无效 |
| 1002 | 未授权访问 |

# 为组件生成 Story
claude "为 src/components/Button.tsx 生成 Storybook story 文件，
包括：
- 默认状态
- 不同尺寸
- 不同变体
- 禁用状态
- 加载状态"

Story 示例

// Button.stories.tsx
import type { Meta, StoryObj } from '@storybook/react';
import { Button } from './Button';

const meta: Meta<typeof Button> = {
    title: 'Components/Button',
    component: Button,
    tags: ['autodocs'],
    argTypes: {
        variant: {
            control: 'select',
            options: ['primary', 'secondary', 'outline']
        },
        size: {
            control: 'select',
            options: ['sm', 'md', 'lg']
        }
    }
};

export default meta;
type Story = StoryObj<typeof Button>;

export const Primary: Story = {
    args: {
        children: 'Primary Button',
        variant: 'primary'
    }
};

export const Secondary: Story = {
    args: {
        children: 'Secondary Button',
        variant: 'secondary'
    }
};

export const Outline: Story = {
    args: {
        children: 'Outline Button',
        variant: 'outline'
    }
};

export const Disabled: Story = {
    args: {
        children: 'Disabled Button',
        disabled: true
    }
};

export const Loading: Story = {
    args: {
        children: 'Loading...',
        loading: true
    }
};

文档维护

文档同步 Hook

{
  "hooks": [
    {
      "id": "sync-readme",
      "event": "PostToolUse",
      "conditions": [
        { "type": "file", "value": "package.json" }
      ],
      "type": "function",
      "function": "async (ctx) => { if (ctx.data.params.path === 'package.json') { const pkg = JSON.parse(ctx.data.params.content); await updateReadmeWithNewDeps(pkg.dependencies); } }"
    }
  ]
}
