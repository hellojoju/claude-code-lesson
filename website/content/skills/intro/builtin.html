内置技能详解

来源说明：本文内容基于 Claude Code 官方文档整理。

可靠性评级：A 官方文档

code-review - 代码审查

功能说明

自动审查代码质量，发现潜在问题并提供改进建议。

使用方法

# 审查单个文件
> /skill run code-review --file src/index.js

# 审查多个文件
> /skill run code-review --files "src/**/*.js"

# 指定审查重点
> /skill run code-review --file src/api.js --focus security

# 自然语言触发
> 审查 src/index.js 的代码质量

审查维度

  • security：安全问题
  • performance：性能问题
  • style：代码风格
  • best-practices：最佳实践
  • complexity：代码复杂度

test-gen - 测试生成

功能说明

自动为代码生成单元测试。

使用方法

# 生成单元测试
> /skill run test-gen --file src/utils.js

# 指定测试框架
> /skill run test-gen --file src/utils.js --framework jest

# 生成集成测试
> /skill run test-gen --file src/api.js --type integration

# 自然语言触发
> 为 src/utils.js 生成测试

支持的测试框架

  • Jest
  • Vitest
  • Mocha
  • Pytest (Python)
  • JUnit (Java)

doc-gen - 文档生成

功能说明

自动生成代码文档。

使用方法

# 生成 API 文档
> /skill run doc-gen --file src/api.js --type api

# 生成 README
> /skill run doc-gen --type readme

# 生成 JSDoc 注释
> /skill run doc-gen --file src/utils.js --format jsdoc

# 自然语言触发
> 为 src/api.js 生成文档

文档类型

  • api：API 文档
  • readme：README 文件
  • jsdoc：JSDoc 注释
  • typedoc：TypeDoc 文档

refactor - 重构辅助

功能说明

辅助代码重构，提供重构建议和自动化重构。

使用方法

# 分析重构机会
> /skill run refactor --file src/index.js --analyze

# 提取函数
> /skill run refactor --file src/index.js --extract-function --name helper

# 重命名
> /skill run refactor --rename oldName newName

# 现代化代码
> /skill run refactor --file src/legacy.js --modernize

重构类型

  • extract-function：提取函数
  • extract-variable：提取变量
  • rename：重命名
  • modernize：现代化语法
  • simplify：简化代码

analyze - 代码分析

功能说明

深度分析代码，提供质量报告。

使用方法

# 分析代码质量
> /skill run analyze --file src/index.js

# 分析依赖关系
> /skill run analyze --type dependencies

# 分析代码复杂度
> /skill run analyze --type complexity

# 分析项目结构
> /skill run analyze --type structure

分析报告示例

代码分析报告
============

文件: src/index.js
行数: 150
函数数: 8
平均函数长度: 18.75 行

复杂度分析:
- 圈复杂度: 12 (建议 < 10)
- 认知复杂度: 15 (建议 < 15)

问题列表:
1. [高] 第 25 行: 可能的空指针引用
2. [中] 第 42 行: 未处理的 Promise
3. [低] 第 15 行: 未使用的变量

debug - 调试辅助

功能说明

辅助调试代码问题。

使用方法

# 分析错误
> /skill run debug --error "TypeError: Cannot read property"

# 分析日志
> /skill run debug --log error.log

# 追踪问题
> /skill run debug --file src/api.js --trace

explain - 代码解释

功能说明

详细解释代码的功能和逻辑。

使用方法

# 解释文件
> /skill run explain --file src/index.js

# 解释函数
> /skill run explain --file src/utils.js --function formatDate

# 解释选中代码
> 解释这段代码的功能

translate - 代码翻译

功能说明

将代码从一种语言翻译到另一种语言。

使用方法

# 翻译到 Python
> /skill run translate --file src/utils.js --to python

# 翻译到 TypeScript
> /skill run translate --file src/utils.js --to typescript

# 自然语言触发
> 把这段 JavaScript 代码翻译成 Python
