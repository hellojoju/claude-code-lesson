文档生成技能案例

来源说明：本文展示一个完整的文档生成技能实现。

可靠性评级：B 社区实践

技能概述

这个文档生成技能可以自动为代码生成文档，支持多种文档格式。

功能特性

  • 支持 JSDoc、TSDoc 注释生成
  • 支持 API 文档生成
  • 支持 README 文件生成
  • 支持多语言文档

技能配置

{
  "name": "doc-gen",
  "version": "2.0.0",
  "description": "智能文档生成技能",
  
  "triggers": {
    "manual": true,
    "keywords": ["生成文档", "写文档", "document", "document this"]
  },
  
  "inputs": {
    "file": {
      "type": "file",
      "description": "源文件"
    },
    "type": {
      "type": "string",
      "options": ["jsdoc", "api", "readme", "changelog"],
      "default": "jsdoc",
      "description": "文档类型"
    },
    "format": {
      "type": "string",
      "options": ["markdown", "html", "json"],
      "default": "markdown",
      "description": "输出格式"
    },
    "language": {
      "type": "string",
      "options": ["zh-CN", "en-US"],
      "default": "zh-CN",
      "description": "文档语言"
    }
  }
}

{{#if (eq type "jsdoc")}}
### JSDoc 注释要求

为每个函数/方法生成 JSDoc 注释：

```javascript
/**
 * 函数描述
 * 
 * @param {类型} 参数名 - 参数描述
 * @returns {类型} 返回值描述
 * @throws {错误类型} 错误描述
 * 
 * @example
 * ```js
 * const result = functionName(arg1, arg2);
 * // result: 期望结果
 * ```
 */
```
{{/if}}

## 输出要求

1. 文档要清晰易懂
2. 示例要完整可运行
3. 参数描述要准确
4. 包含必要的注意事项

使用方法

生成 JSDoc

# 为文件生成 JSDoc 注释
> /skill run doc-gen --file src/utils.js --type jsdoc

API 文档示例

# 用户 API 文档

## 获取用户列表

**描述**: 获取所有用户的列表

**方法**: GET

**路径**: /api/users

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| page | number | 否 | 页码，默认 1 |
| limit | number | 否 | 每页数量，默认 10 |
| search | string | 否 | 搜索关键词 |

### 请求示例

```http
GET /api/users?page=1&limit=10
Authorization: Bearer {token}
```

### 响应参数

| 参数名 | 类型 | 描述 |
|--------|------|------|
| users | User[] | 用户列表 |
| total | number | 总数量 |
| page | number | 当前页码 |

### 响应示例

```json
{
  "users": [
    {
      "id": "1",
      "name": "张三",
      "email": "zhangsan@example.com"
    }
  ],
  "total": 100,
  "page": 1
}
```

### 错误码

| 错误码 | 描述 |
|--------|------|
| 401 | 未授权 |
| 403 | 无权限 |
| 500 | 服务器错误 |

---

## 创建用户

**描述**: 创建新用户

**方法**: POST

**路径**: /api/users

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| name | string | 是 | 用户名 |
| email | string | 是 | 邮箱地址 |
| password | string | 是 | 密码 |

### 请求示例

```json
{
  "name": "张三",
  "email": "zhangsan@example.com",
  "password": "password123"
}
```

### 响应示例

```json
{
  "id": "1",
  "name": "张三",
  "email": "zhangsan@example.com",
  "createdAt": "2026-04-04T10:00:00Z"
}
```
